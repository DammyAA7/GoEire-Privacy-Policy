Common Issues

Departures not loading

Ensure you have an active internet connection. Departure data refreshes every 15 seconds — pull down or tap a stop again to retry.

Location not working

GoEire needs location access to show nearby stops. Go to Settings > GoEire > Location and select "While Using the App".

Stop missing or incorrect

Our stop data covers all public transit stops in Ireland. If you notice a missing or misplaced stop, please email us with the stop name or code and we'll investigate.

In-App Feedback

You can also submit bug reports and feature requests directly from the app via Settings > Send Feedback.

App Version

Make sure you're running the latest version of GoEire from the App Store for the best experience.
